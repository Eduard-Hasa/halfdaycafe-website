Plugin documentation

Add styles in <head> tag:

<link rel="stylesheet" href="assets/vendors/slick/slick.css">
<link rel="stylesheet" href="assets/vendors/slick/slick-theme.css">

Add script before closing <body> tag:

<script src="assets/vendors/slick/slick.min.js"></script>

Equal heights

Add class .slick-slider--equal-height to your slider. It will make .slick-track inside slider as flexbox container and stretch .slick-slide elements, so they will be same height.

Gallery

This is a nice and compact way to display a photo gallery. As it's already integrated with Fancybox, don't forget to include its style and script files on your page.

This slider has requirements for image sizes:

  • Normal image: 480px x 335px
  • Wide image: 480px * 2 + $grid-gutter-width (30px default) x 335px
  • Tall image: 480px x 335 * 2 + $grid-gutter-width (30px default)

If you want to have other image sizes, follow these size ratios.
