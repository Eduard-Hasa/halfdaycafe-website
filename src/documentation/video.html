Video

Fika Only

Video players are handled by the Plyr and Jarallax libraries. Plyr is a simple, lightweight, accessible and customizable HTML5, YouTube and Vimeo media player that supports modern browsers. If you want to have a background video Jarallax will come to help. It also provides a nice parallax scrolling effect.

Plyr.js video players

Embed videos from Youtube, Vimeo or local storage using the Plyr plugin library. Fika uses .plyr-video class for initialization.

Plugin documentation

Add styles in <head> tag:

<link rel="stylesheet" href="assets/vendors/plyr/plyr.min.css">

Add script before closing <body> tag:

<script src="assets/vendors/plyr/plyr.min.js"></script>

Inline video local example

<!-- Video box -->
    <div class="rounded overflow-hidden">
        <video poster="../assets/images/video-preview/counter.jpg" id="player" class="plyr-video">
            <source src="../assets/videos/Counter.mp4" type="video/mp4" />
        </video>
    </div>

Inline Video Embedded


    <!-- Video box -->
    <div class="rounded overflow-hidden">
        <div class="plyr-video" data-plyr-provider="vimeo" data-plyr-embed-id="248303052"></div>
    </div>

Jarallax background video

The Jarallax plugin also handles fullscreen video backgrounds from Vimeo, Youtube or local video sources.

Plugin documentation

Add styles in <head> tag:

<link rel="stylesheet" href="assets/vendors/jarallax/jarallax.css">

Add scripts before closing <body> tag:

<script src="assets/vendors/jarallax/jarallax.min.js"></script>
<script src="assets/vendors/jarallax/jarallax-videos.min.js"></script>

Simply add data-jarallax-video attribute with Vimeo, Youtube or local video link. Note, that to disable parallax effect you need to add data-speed="1" attribute. If you need to make a text easier to read, consider using overlay utility to darken the background video.
