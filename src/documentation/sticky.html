Sticky

Fika Only

Pure JavaScript plugin for making smart and high performance sticky sidebars.

Plugin documentation

Add script before closing <body> tag:

<script src="assets/vendors/sticky-sidebar/jquery.sticky-sidebar.min.js"></script>

Basic Example

Scrollable Sticky Element

<div class="container">
    <div data-sticky='{options}'>
        <div class="inner">Sticky Content</div>
    </div>
</div>
