Wide Buttons

Fika Only

If you think, that standard Bootstrap’s buttons look a bit compressed, use .btn-wide modifier to make them wider.

<button type="button" class="btn btn-wide btn-primary">Simple</button>
<button type="button" class="btn btn-wide btn-lg btn-primary">Simple</button>
<button type="button" class="btn btn-arrow btn-wide btn-dark">Arrow</button>
<button type="button" class="btn btn-arrow btn-wide btn-lg btn-dark">Arrow</button>

Hero Button

Fika Only

In some cases, you might need your button to stand out, use .btn-hero modifier to do that.

<button type="button" class="btn btn-hero btn-primary">Hero</button>
<button type="button" class="btn btn-hero btn-wide btn-primary">Hero</button>
