Image

Fika comes with a bunch of useful utilities which you can use to adjust background image as you want. Start by using .bg-img class. It will set background-repeat property to no-repeat, background-size to auto and background-position to center center.

<div class="bg-img" style="background-image: url(...)"></div>

Image size

Use these modifiers to control background image size.

.bg-img--cover - cover;

.bg-img--100 - 100% 100%;

.bg-img--horizontal-100 - horizontal - 100%, vertical - auto;

.bg-img--vertical-100 - horizontal - auto, vertical - 100%;

Image position

Use these modifiers to control background image position.

.bg-img--top - horizontal - center, vertical - top

.bg-img--top-right - horizontal - right, vertical - top

.bg-img--top-left - horizontal - left, vertical - top

.bg-img--bottom - horizontal - center, vertical - bottom

.bg-img--bottom-right - horizontal - right, vertical - bottom

.bg-img--bottom-left - horizontal - left, vertical - bottom;

.bg-img--center - horizontal - center, vertical center

.bg-img--center-right - horizontal - right, vertical - center

.bg-img--center-left - horizontal - left, vertical - center;
