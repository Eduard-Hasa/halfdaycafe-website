Map

Fika Only

Adds an interactive map powered by Google Maps.

Plugin documentation

Get API Key

Add script before closing <body> tag:

<script src="//maps.googleapis.com/maps/api/js?key=your_api_key"></script>
  • .google-map - initializes the map.
  • data-lat="59.3169" data-lng="18.0605" - latitude and longitude for the center point. Longitude is a a number between -180 and 180. Latitude is a number between -90 and 90.
  • data-icon="..." - pin image.
  • data-zoom="15" - zoom level; a number between 0 and 20. Fractional zoom levels will be rounded to two decimal places.
<div class="google-map" data-lat="59.3169" data-lng="18.0605" data-icon="../assets/images/icons/custom/location-1.svg" data-zoom="15"></div>
