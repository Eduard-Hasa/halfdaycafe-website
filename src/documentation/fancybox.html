Fancybox

Fika Only

JavaScript lightbox library for presenting various types of media. Responsive, touch-enabled and customizable. Fika uses Extended Commercial license, so you do not need to purchase your own license â€” as long as you are not developing own commercial products with fancybox.

Plugin documentation

Add styles in <head> tag:

<link rel="stylesheet" href="assets/vendors/fancybox/jquery.fancybox.min.css">

Add script before closing <body> tag:

<script src="assets/vendors/fancybox/jquery.fancybox.min.js"></script>

Common use

The most basic way to use fancybox is by adding the data-fancybox attribute to your element. This will automatically bind click event that will start fancybox. Use href or data-src attribute to specify source of your content.

Fancybox gallery

If you have a group of items, you can use the same attribute data-fancybox value for each of them to create a gallery. Each group should have a unique value.

<div class="row">
    <div class="col-6 col-md-4 col-lg-3">
        <a href="..." data-fancybox="gallery-1" class="d-block my-2 lightbox-hover-scale">
            <img src="..." alt="..." class="img-fluid">
        </a>
    </div>
    <div class="col-6 col-md-4 col-lg-3">
        <a href="..." data-fancybox="gallery-1" class="d-block my-2 lightbox-hover-scale">
            <img src="..." alt="..." class="img-fluid">
        </a>
    </div>

Fancybox gallery with Slick carousel

If you want to use fancybox elements inside slick slider, use data-slick-fancybox to group images instead of data-fancybox. Also, add data-fancybox-index attribute to every item to avoid duplications, when slick carousel is in an infinity mode.

        "arrows": false, 
        "dots": false, 
        "slidesToShow": 6,

                "settings": {
                    "slidesToShow": 4
