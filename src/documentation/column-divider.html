Column Dividers

Fika Only

You can add nice dividers between columns by adding .divided-columns class to a row. It also affects <hr> tag inside that row, which allows you to beautifully divide content inside a column.

You can use color modifiers, point after which breakpoint dividers should be removed, use a standard width or make them thicker.

  • .divided-columns - adds right border to each column except the last one
  • .divided-columns--sm - tells to show dividers when a screen width is bigger than sm breakpoint
  • .divided-columns--dark - changes divider's color to a dark one
  • .divided-columns--thick - makes dividers thicker
