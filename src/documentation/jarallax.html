Jarallax

Fika Only

Smooth parallax scrolling effect for background images, videos and inline elements.

Plugin documentation

Add scripts before closing <body> tag:

<script src="assets/vendors/jarallax/jarallax.min.js"></script>
<script src="assets/vendors/jarallax/jarallax-elements.min.js"></script>

Background image

We recommend you use Jarallax with background images to ensure compatibility with IE.

<div class="jarallax" data-speed="0.5" style="background-image: url(../assets/images/sections-bg/home-intro-bg-2.jpg);"></div>
