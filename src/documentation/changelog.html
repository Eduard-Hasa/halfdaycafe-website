Important changes in build process.

We have changed our build process, in order to make it more effective. Now all source files will be placed inside src folder, but after compilation, all files will be moved into dist folder.

Just follow these simple steps:

  1. Create src folder;
  2. Move your assets folder into src folder;
  3. Move your documentation folder into src folder;
  4. Move all *.html files from root into src folder;
  5. Run npm install command in your console to update packages;
  6. Run gulp command in your console when you need to edit files;
  7. Run gulp build command in your console when you need to build your site for producation.
